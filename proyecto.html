Sobre el Proyecto

El proyecto “Implementación de un sistema web para la publicación de eventos utilizando drones” busca mejorar la visualización, documentación y difusión de las actividades institucionales mediante el uso de tecnología aérea y plataformas web.

  • Uso de drones para captación de imágenes y videos
  • Plataforma web accesible para la comunidad educativa
  • Difusión organizada de eventos institucionales
